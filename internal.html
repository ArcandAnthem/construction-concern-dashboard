Root Cause Analysis
Problem #1 — Hook Failure

Facebook reaches 200 people per post but gets zero reactions. Content is being distributed but not stopping the scroll. The post format (lifestyle image + brochure caption) is not compelling enough for mid-feed behavior.

Problem #2 — No Audience Building

The system has been a publishing system only. No community engagement (commenting, following, participating), which means the algorithm receives no engagement signals to reward with wider distribution.

Problem #3 — Format Mismatch

Single static images are the lowest-performing format on both platforms. Carousels get 3× more algorithmic distribution on Instagram. No carousel posts have been published yet.

What's Actually Working

List-format posts (confirmed by data), fire safety SEO content (168 views — #1 by far), and Facebook organic distribution (200 reach/post is strong for a new page). The foundation is there.
