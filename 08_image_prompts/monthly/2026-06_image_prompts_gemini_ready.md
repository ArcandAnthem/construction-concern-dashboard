# AI-Ready Image Prompts — Construction Concern — June 2026

## How to Use This File
- Paste each "Image Prompt" directly into **ChatGPT** (GPT-4o) or **Gemini** — both generate images from these prompts
- Each image can be reused across **all four platforms** (Instagram, Facebook, LinkedIn, Threads)
- Posts marked **USE REAL PHOTO** — skip the AI prompt and pull the named file from `Photos_Before and After/` folder instead
- Carousel posts (Posts 2, 5, 8) need one cover image + the text slides built separately in Canva or similar

## Audience Home Style Note
All exterior prompts should reflect the **actual homes in Santa Clarita, Newhall, and the San Fernando Valley**: stucco exteriors, tile roofs, traditional double-hung or slider windows, drought-tolerant front yard landscaping, and suburban tract-style architecture from the 1980s–2000s. Not modern, architect-designed, or high-end contemporary homes.
