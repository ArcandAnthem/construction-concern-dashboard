# Wix Newsletter Template — Construction Concern
**Reusable Monthly Template | Arc & Anthem**

## Recommended Send Schedule

| Newsletter | Day | Time | Notes |
|---|---|---|---|
| Newsletter #1 | **Tuesday** | **10:00am PST** | First week of the month |
| Newsletter #2 | **Tuesday** | **10:00am PST** | Third week of the month |

**Why Tuesday at 10am:** Highest B2C email open rates. After morning commute, before midday. Avoid Mondays (inbox overload) and Fridays (weekend distraction).

## Before You Start — Image Prompts

Generate **two images** per newsletter before building in Wix:
1. **Hero image** — full-width top of newsletter
2. **Mid-newsletter image** — smaller, placed just before the Quick Tip callout

Use ChatGPT (GPT-4o image generation) or the API script to generate both before opening Wix.

### How to adjust prompts for any topic:
Replace the scene description. Keep all other instructions (dimensions, style, no text) exactly the same.

**Hero image (600×400) scene swaps:**
- **Windows:** bright living room, large updated window frames, natural afternoon light
- **HVAC / Comfort:** comfortable family room, ceiling fan visible, open floor plan
- **Siding:** freshly replaced siding on a SoCal home, clean curb appeal, clear sky
- **Kitchen remodel:** updated kitchen, warm cabinetry, stone counters, natural light
- **Bathroom remodel:** clean modern bathroom, subway tile, natural stone, bright vanity
- **Solar:** rooftop solar panels on a SoCal home, clear sky, clean install

**Mid-newsletter image (600×300) scene swaps:**
- **Windows:** close-up of window frame seal, weatherstripping, or glass pane detail
- **HVAC:** close-up of HVAC filter, vent grille, or thermostat on a wall
- **Siding:** close-up of fiber cement or stucco siding texture, clean install
- **Kitchen:** close-up of cabinet hardware, countertop edge, or backsplash tile
- **Bathroom:** close-up of faucet, tile grout, or vanity detail
- **Solar:** close-up of solar panel edge on a roof, blue sky background

## Wix Email Builder — Block-by-Block Setup

Open Wix Email Marketing → Create Campaign → Blank Template.
Follow each block in order. Save the layout as a template when done so you only set this up once.

---

### Block 1 — Header / Logo
**Block type:** Image block (full width)
**Settings:**
- Upload the Construction Concern logo (centered)
- Background color: `#D4C5A9` (warm sand)
- Padding: 20px top, 20px bottom
- Width: Full width

> First time only: Upload the logo and save this block to your Saved Blocks in Wix.

---

### Block 2 — Hero Image
**Block type:** Image block (full width)
**Settings:**
- Upload the hero image generated from the **Hero Image Prompt** above (600px × 400px)
- No caption
- Padding: 0px all sides
- Wix auto-fits to full 600px width — use the 600×400 image and it will display cleanly

---

### Block 3 — Subject Echo / Issue Label
**Block type:** Text block
**Settings:**
- Background color: `#f5f2ed` (off-white / light sand)
- Padding: 16px all sides
- Font: Arial or Helvetica | 11px | uppercase | letter-spacing | color `#5a4f3f`
- Text to type: `CONSTRUCTION CONCERN  ·  [MONTH YEAR]  ·  HOME INSIGHTS`

---

### Block 4 — Opening Hook
**Block type:** Text block
**Settings:**
- Background: white `#ffffff`
- Padding: 28px left/right, 20px top/bottom
- Font: Georgia or serif | 17px | color `#2c2520` | line height 1.7

**Fill in:**
```
[OPENING HOOK — 2 to 4 sentences. State the seasonal moment or homeowner problem.
Be direct and calm. This is not a sales opener. Example:
"June is when LADWP bills start to climb. If your home isn't efficient — even a little —
the difference can be $50–100 a month in additional cooling costs."]
```

---

### Block 5 — Section Divider
**Block type:** Divider
**Settings:**
- Color: `#D4C5A9`
- Height: 1px
- Width: 80%
- Padding: 10px top and bottom

---

### Block 6 — Section Title
**Block type:** Text block
**Settings:**
- Background: `#f5f2ed`
- Padding: 16px all sides
- Font: Arial | 13px | bold | uppercase | color `#5a4f3f`

**Fill in:**
```
[SECTION TITLE — e.g., "YOUR JUNE HOME ENERGY CHECKLIST" or "WHAT TO LOOK AT THIS MONTH"]
```

---

### Block 7 — Main Content (Checklist or Numbered Points)
**Block type:** Text block
**Settings:**
- Background: white `#ffffff`
- Padding: 28px left/right, 16px top/bottom
- Font: Georgia or serif | 15px | color `#2c2520` | line height 1.8

**Fill in (repeat pattern for each point — typically 3 to 5 points):**
```
✅ [BOLD POINT TITLE — short, 4–7 words]
[2–3 sentence explanation. Specific, helpful, non-salesy. Written for a homeowner who wants honest guidance.]

✅ [BOLD POINT TITLE]
[2–3 sentence explanation.]

✅ [BOLD POINT TITLE]
[2–3 sentence explanation.]
```

> If the newsletter is not a checklist format, use numbered sections (Phase 1, Phase 2…)
> or short subheads with a paragraph under each. Same spacing and font rules apply.

---

### Block 7B — Mid-Newsletter Supporting Image
**Block type:** Image block (full width)
**Settings:**
- Upload the mid-newsletter image generated from the **Mid-Newsletter Image Prompt** above (600px × 300px)
- No caption
- Padding: 0px top, 8px bottom
- Place this block directly above the Quick Tip Callout (Block 8)

---

### Block 8 — Quick Tip Callout
**Block type:** Text block
**Settings:**
- Background: `#f9f7f3`
- Left border style: add a 4px left border using Wix column border tool, color `#D4C5A9`
- Padding: 16px all sides
- Font: Georgia | 14px | italic for tip label | color `#3d3530`

**Fill in:**
```
Quick tip: [ONE practical tip — 1 to 2 sentences. Homeowner-specific.
Example: "The best time to book summer home improvement work is before it gets hot."]
```

---

### Block 9 — Blog Reference
**Block type:** Text block (or Button + Text combo)
**Settings:**
- Background: white `#ffffff`
- Padding: 20px all sides
- Font: Georgia | 15px | color `#2c2520`

**Fill in:**
```
This month on the Construction Concern blog: [BLOG POST TITLE IN ITALICS]

Read it at constructionconcern.com
```

> Optional: Add a Wix Button block below this text
> Button label: "Read the Blog"
> Button color: `#D4C5A9` | Text color: `#2c2520` | Style: rounded | Centered

---

### Block 10 — CTA Section
**Block type:** Text block + Button block
**Settings:**
- Background: `#f5f2ed`
- Padding: 28px all sides
- Font: Georgia | 15px | color `#2c2520`

**Fill in:**
```
[CLOSING SENTENCE — one line. Connect the content to the next step.
Example: "If any of the checklist items above match what you're seeing at home, we'd love to help."]

Free estimates are available for all projects — windows, HVAC, siding, and more.
```

**Button:**
- Label: `Request a Free Estimate`
- Link: `constructionconcern.com/free-estimate`
- Color: `#D4C5A9` | Text: `#2c2520` | Style: rounded | Centered

---

## Saving as a Wix Template

After building this the first time:
1. In the Wix Email Editor, click **Save as Template** (top right menu)
2. Name it: `Construction Concern — Monthly Newsletter`
3. Each month, open this template → duplicate it → fill in the new content
4. Never edit the saved template directly — always duplicate first
