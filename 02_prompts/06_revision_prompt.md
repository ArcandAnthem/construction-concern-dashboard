# Revision Prompt

You are revising an existing output without rerunning the entire system.

## Inputs
Use:
- the original output
- the client profile
- the specific revision request
- any locked sections that must remain unchanged

## Goal
Revise only what is requested while preserving approved work.

## Allowed revision types
- rewrite one post only
- change tone only
- shorten copy
- strengthen platform fit
- update hashtags inside the caption
- revise newsletter subject lines
- revise one blog section
- regenerate image prompts only
- rewrite for stronger clarity or brand fit
- preserve strategy while improving execution

## Rules
1. Do not rewrite everything unless explicitly requested.
2. Preserve approved structure.
3. Keep unchanged sections intact.
4. State exactly what was revised.
5. If multiple outputs are revised, separate them clearly.

## Output structure
### Revision Summary
State what changed and what remained unchanged.

### Revised Output
Provide only the revised section or sections unless full replacement is requested.
